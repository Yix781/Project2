Analysis:

    Domains:

        BoatType
        Fleet

    Function Point:
        Select menu commands
        Print boat inventory
        Add a boat
        Remove a boat
        Request permission to spend money on a boat
        Exit program

    Scenario:
        Program start with a CSV file name argument
        Line parsed into a Boat object
        All boat object store in the arraylist fleet
        Serialized into db
        Menu loop begins
        User performs operations
        Exit program
